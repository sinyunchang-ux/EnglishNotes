操作指引

安裝、日常使用、CSV 匯入 的完整步驟說明

安裝 APK

1

下載 APK

前往 GitHub Actions 頁面下載最新編譯結果：

https://github.com/sinyunchang-ux/EnglishNotes/actions

點進最新一次綠色 ✅ 的 Build APK，在頁面底部 Artifacts 區塊下載 EnglishNotes-debug.zip，解壓縮得到 app-debug.apk。

2

傳到手機

任何方式皆可：

  • 用 Line 傳給自己（傳送「檔案」而非圖片）
  • 用 USB 線傳輸
  • 用 Google Drive 上傳再在手機下載
3

允許安裝未知來源

點開 APK 後，Android 會詢問是否允許安裝，選擇 「允許」 即可。安裝完成後直接開啟。

ℹ️
日後更新：直接安裝新版 APK 覆蓋即可，資料完整保留，不需解除安裝舊版。

新增筆記

1

點右上角「＋ 新增」按鈕

進入新增筆記對話框。

2

填寫欄位

  • 英文句子（必填）
  • 中文翻譯（選填）

「新增日期」會自動帶入今天日期，不需手動填寫。

3

點「新增」完成

筆記立即出現在列表第一行。

錄音

1

點該筆記的麥克風 🎙 按鈕

首次使用會請求麥克風權限，請點「允許」。

2

在錄音對話框點「開始錄音」

進度條會顯示目前錄音時長，最長 180 秒，時間到自動停止。

3

點「停止錄音」儲存

錄音日期自動記錄為今天，月曆上該日期變為綠色。

⚠️
重新錄音會覆蓋舊錄音，舊錄音無法復原。

分享錄音到 Line

✅
點列表中的 分享 📤 按鈕 → 系統分享表出現 → 選 Line → 選擇對話對象 → 傳送。

CSV 匯入

可一次大量匯入筆記，不需逐條輸入。

📄
CSV 格式：第一欄英文，第二欄中文
支援有標題列（自動跳過）或無標題列皆可。 
# 範例 CSV 內容（存成 .csv 檔）
English,Chinese
I love learning English.,我喜歡學英文。
The weather is nice today.,今天天氣很好。
Could you say that again?,你可以再說一次嗎？
1

點「匯入 CSV」按鈕

開啟手機檔案選擇器。

2

選擇 .csv 檔

可從 Google Drive、下載資料夾等任何位置選取。

3

自動匯入完成

所有筆記以今天為「新增日期」批次加入，無需手動填日期。

資料庫 Schema

@Entity(tableName = "notes")
data class NoteEntity(
    @PrimaryKey(autoGenerate = true)
    val id: Long = 0,
    val english: String,          // 英文句子
    val chinese: String,          // 中文翻譯
    val recordingPath: String?,   // 錄音檔路徑（null = 未錄音）
    val addedDate: String,        // 新增日期 "2026-06-11"
    val recordingDate: String?    // 錄音日期 "2026-06-11"（null = 未錄音）
)

功能記錄

已實作功能、版本更新記錄、待辦清單

已實作功能

📊 表格列表視圖

類 Google 試算表呈現方式，顯示英文、中文、錄音操作、新增日期、錄音日期五欄；支援橫向滑動。

📅 月曆視圖

以月為單位顯示筆記分佈。有錄音的日期顯示綠色，有筆記但無錄音顯示藍色，點擊日期展開當天筆記列表。

🎙 錄音（最長 180 秒）

每筆筆記可附一段錄音，進度條顯示時長，到達 180 秒自動停止。錄音格式 M4A，儲存於 App 內部目錄。

▶️ 播放錄音

在列表直接點播放鍵收聽錄音，播放中顯示停止按鈕，播完自動重置。

📤 分享錄音到 Line

透過 Android 系統分享表，可分享到 Line、訊息、Email 等任意 App；分享內容包含錄音檔及英文、中文文字。

➕ 單條新增筆記

彈出對話框逐欄填寫，英文必填、中文選填。新增日期自動記錄當天，格式 2026-06-11。

📥 CSV 批次匯入

選取 CSV 檔後自動解析，第一欄英文、第二欄中文，支援有/無標題列，批次新增不佔用操作時間。

✏️ 編輯筆記

可修改任一筆記的英文與中文內容，錄音獨立保存不受影響。

🗑️ 刪除筆記

刪除時同步刪除對應錄音檔，釋放儲存空間；刪除前有確認對話框防止誤觸。

📆 自動日期記錄

新增日期與錄音日期皆自動帶入操作當天，格式統一為 YYYY-MM-DD，無需手動輸入。
